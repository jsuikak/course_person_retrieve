# 统一特征提取接口说明

## 1. 目标

提供一个统一接口，同时处理两类输入：

1. `face` 图像：调用 ArcFace 提取特征。
2. `person` 图像：调用 ResNet 提取特征。

接口输入统一为内存图像（`np.ndarray`，BGR），输出统一为特征向量，并支持 batch（适合视频逐帧提取）。

---

## 2. 代码位置

- 接口与实现：`src/tools/feature_extractor.py`

---

## 3. 核心类型

### 3.1 模式枚举

`FeatureMode`

1. `FeatureMode.FACE`（值：`"face"`）
2. `FeatureMode.PERSON`（值：`"person"`）

### 3.2 提取器配置

`FeatureExtractorConfig`

关键字段：

1. ArcFace 相关：`arcface_weight_path`, `detect_face`, `face_flip_test`
2. ResNet 相关：`resnet_backbone`, `resnet_pretrained`, `resnet_weight_path`
3. 通用：`device`

---

## 4. 接口定义

## 4.1 单条提取

```python
extract(mode: FeatureMode, image_bgr: np.ndarray) -> Optional[np.ndarray]
```

行为：

1. `mode=face`：走 ArcFace。
2. `mode=person`：走 ResNet。
3. 失败返回 `None`，成功返回 `float32` 特征向量。

## 4.2 批量提取（保留输入顺序）

```python
extract_batch(mode: FeatureMode, images_bgr: Sequence[np.ndarray]) -> List[Optional[np.ndarray]]
```

行为：

1. 输出列表与输入等长。
2. 每个位置对应一个特征或 `None`（该样本失败）。

适合场景：

1. 视频帧批量提取。
2. 数据集批量建索引。

## 4.3 批量提取（矩阵输出）

```python
extract_batch_matrix(mode: FeatureMode, images_bgr: Sequence[np.ndarray]) -> Tuple[np.ndarray, List[int]]
```

行为：

1. 返回 `features_matrix`（只包含成功样本）和 `valid_indices`。

---

## 5. 最小示例

```python
from src.tools.feature_extractor import (
    FeatureExtractorConfig,
    FeatureExtractor,
    FeatureMode,
)
import cv2

extractor = FeatureExtractor(
    FeatureExtractorConfig(
        arcface_weight_path="./models/weights/arcface.pt",
        device="cpu",
        resnet_backbone="resnet50",
        resnet_pretrained=False,
    )
)

# 单条
face_img = cv2.imread("data/query_face.jpg")
person_img = cv2.imread("data/query_person.jpg")
face_feat = extractor.extract(mode=FeatureMode.FACE, image_bgr=face_img)
person_feat = extractor.extract(mode=FeatureMode.PERSON, image_bgr=person_img)

# batch（单一 mode）
frames = [cv2.imread("a.jpg"), cv2.imread("b.jpg")]
batch_feats = extractor.extract_batch(mode=FeatureMode.PERSON, images_bgr=frames)
```

---

## 6. 输入输出汇总表

| 接口 | 输入 | 输出 | 说明 |
|---|---|---|---|
| `extract(mode, image_bgr)` | `FeatureMode` + 单张内存图像 `np.ndarray(BGR)` | `Optional[np.ndarray]` | 单条提取；失败返回 `None` |
| `extract_batch(mode, images_bgr)` | `FeatureMode` + `Sequence[np.ndarray(BGR)]` | `List[Optional[np.ndarray]]` | 批量提取；与输入等长，失败位为 `None` |
| `extract_batch_matrix(mode, images_bgr)` | `FeatureMode` + `Sequence[np.ndarray(BGR)]` | `Tuple[np.ndarray, List[int]]` | 压缩输出；`np.ndarray` 仅保留成功特征，`List[int]` 是有效输入下标 |
