# 人脸特征提取封装接口说明

## 1. 目标

提供一个最简封装接口，完成以下流程：

1. 获取图像（内存或磁盘）。
2. 用 MTCNN 检测人脸框。
3. 对每个检测框做人脸裁剪。
4. 将裁剪图送入人脸特征提取接口（ArcFace）。
5. 输出人脸特征结果。

接口同时支持：

1. 单图提取。
2. 图像库提取。
3. 视频库提取。

---

## 2. 代码位置

- `src/face_feature_pipeline.py`

---

## 3. 核心类型

### 3.1 配置

`FaceFeaturePipelineConfig`

1. `arcface_weight_path: str`
2. `device: str = "cpu"`
3. `face_flip_test: bool = True`
4. `face_blur_threshold: float = 0.0`
5. `face_min_size: int = 0`
6. `mtcnn_min_face_size: float = 20.0`
7. `mtcnn_thresholds: tuple[float, float, float] = (0.6, 0.7, 0.8)`
8. `mtcnn_nms_thresholds: tuple[float, float, float] = (0.7, 0.7, 0.7)`

### 3.2 单条记录

`FaceFeatureRecord`

1. `source_type: str`（`image` 或 `video`）
2. `source_name: str`
3. `frame_index: int`（图片为 `-1`）
4. `face_index: int`
5. `bbox: FaceBox`
6. `feature: np.ndarray`

### 3.3 返回值

`FaceFeatureBundle` 类型

1. `records: List[FaceFeatureRecord]`
2. `feature_matrix() -> np.ndarray`
3. `dump(output_dir, prefix) -> dict[str, str]` 落盘函数，保存为文件

---

## 4. 接口定义

### 4.1 单图提取（内存/磁盘兼容）

```python
extract_image(
    image_bgr: Optional[np.ndarray] = None,
    image_path: Optional[str] = None,
    source_name: Optional[str] = None,
) -> FaceFeatureBundle
```

约束：

1. `image_bgr` 与 `image_path` 必须二选一。

### 4.2 内存图片批量提取

```python
extract_images(images_bgr: Sequence[np.ndarray], source_prefix: str = "memory_image") -> FaceFeatureBundle
```

### 4.3 图像库提取（磁盘）

```python
extract_image_library(
    image_paths: Optional[Sequence[str]] = None,
    image_dir: Optional[str] = None,
) -> FaceFeatureBundle
```

### 4.4 单视频提取（磁盘）

```python
extract_video(video_path: str, sample_fps: float = 1.0) -> FaceFeatureBundle
```

### 4.5 内存视频帧提取

```python
extract_video_frames(frames_bgr: Sequence[np.ndarray], source_name: str = "memory_video") -> FaceFeatureBundle
```

### 4.6 视频库提取（磁盘）

```python
extract_video_library(
    video_paths: Optional[Sequence[str]] = None,
    video_dir: Optional[str] = None,
    sample_fps: float = 1.0,
) -> FaceFeatureBundle
```

---

## 5. 落盘接口

`FaceFeatureBundle.dump(output_dir, prefix)`

会写出：

1. `{prefix}_features.npy`：特征矩阵。
2. `{prefix}_meta.csv`：每行特征的来源与检测框元数据。
3. `{prefix}_info.json`：摘要信息（数量、维度、文件路径）。

---

## 6. 输入输出汇总表

| 接口 | 输入 | 输出 | 说明 |
|---|---|---|---|
| `extract_image` | 单图（内存或磁盘） | `FaceFeatureBundle` | 单图的人脸特征提取 |
| `extract_images` | 内存图像序列 | `FaceFeatureBundle` | 兼容内存图库 |
| `extract_image_library` | 图像路径列表或目录 | `FaceFeatureBundle` | 兼容磁盘图库 |
| `extract_video` | 单个视频路径 | `FaceFeatureBundle` | 采样帧后提取 |
| `extract_video_frames` | 内存帧序列 | `FaceFeatureBundle` | 兼容内存视频 |
| `extract_video_library` | 视频路径列表或目录 | `FaceFeatureBundle` | 兼容磁盘视频库 |
| `FaceFeatureBundle.dump` | 输出目录 + 前缀 | 文件路径字典 | 建索引场景落盘 |

---

## 7. 最小示例

```python
from src.face_feature_pipeline import FaceFeaturePipeline, FaceFeaturePipelineConfig

pipeline = FaceFeaturePipeline(
    FaceFeaturePipelineConfig(
        arcface_weight_path="./models/weights/arcface.pt",
        device="cpu",
    )
)

bundle = pipeline.extract_image(image_path="data/example.jpg")
paths = bundle.dump(output_dir="outputs/indexes", prefix="gallery")
print(paths)
```

---

## 8. 类信息速览

| 类名 | 主要字段/方法 | 含义 |
|---|---|---|
| `FaceFeaturePipelineConfig` | `arcface_weight_path`, `device`, `face_flip_test`, `mtcnn_min_face_size` 等 | 流程配置（ArcFace + MTCNN） |
| `FaceFeaturePipeline` | `extract_image`, `extract_images`, `extract_image_library`, `extract_video`, `extract_video_frames`, `extract_video_library` | 封装后的统一提取入口 |
| `FaceBox` | `x, y, w, h` | 人脸检测框坐标（左上角 + 宽高） |
| `FaceFeatureRecord` | `source_type`, `source_name`, `frame_index`, `face_index`, `bbox`, `feature` | 单张人脸的完整记录 |
| `FaceFeatureBundle` | `records`, `feature_matrix()`, `dump()` | 一次提取任务的结果集合 |

## 9. 各种输入情况示例

### 9.1 单图（内存输入）

```python
import cv2
from src.face_feature_pipeline import FaceFeaturePipeline, FaceFeaturePipelineConfig

pipeline = FaceFeaturePipeline(
    FaceFeaturePipelineConfig(arcface_weight_path="./models/weights/arcface.pt", device="cpu")
)

image = cv2.imread("data/example.jpg")
bundle = pipeline.extract_image(image_bgr=image, source_name="mem_case_001")

print(len(bundle.records))             # 检测到的人脸数量
print(bundle.records[0].bbox)          # FaceBox(x, y, w, h)
print(bundle.records[0].feature.shape) # (512,)
```

### 9.2 单图（磁盘输入）

```python
bundle = pipeline.extract_image(image_path="data/example.jpg")
print(bundle.records[0].source_type)   # image
print(bundle.records[0].source_name)   # data/example.jpg
print(bundle.records[0].frame_index)   # -1（图片固定为 -1）
```

### 9.3 图像库（磁盘目录）

```python
bundle = pipeline.extract_image_library(image_dir="data/LFW")
features = bundle.feature_matrix()
print(features.shape)  # (N, 512)
```

### 9.4 图像库（内存列表）

```python
import cv2
imgs = [cv2.imread("a.jpg"), cv2.imread("b.jpg"), cv2.imread("c.jpg")]
bundle = pipeline.extract_images(imgs, source_prefix="gallery_mem")
print(len(bundle.records))
```

### 9.5 视频库（单视频路径）

```python
bundle = pipeline.extract_video(video_path="data/yoasobi_TFT/short.mp4", sample_fps=1.0)
print(len(bundle.records))                 # 所有采样帧中检测到的人脸总数
print(bundle.records[0].frame_index)       # 人脸对应帧号
print(bundle.records[0].source_type)       # video
```

### 9.6 视频库（内存帧序列）

```python
# frames 是提前读取到内存的 BGR 帧列表
bundle = pipeline.extract_video_frames(frames_bgr=frames, source_name="mem_video_001")
print(bundle.records[0].source_name)       # mem_video_001
```

### 9.7 建索引落盘

```python
bundle = pipeline.extract_image_library(image_dir="data/LFW")
paths = bundle.dump(output_dir="outputs/indexes", prefix="face_gallery")
print(paths)
# {
#   "features_path": ".../face_gallery_features.npy",
#   "meta_path": ".../face_gallery_meta.csv",
#   "info_path": ".../face_gallery_info.json"
# }
```
