# 统一特征匹配接口说明

## 1. 目标

提供一个统一接口，完成查询特征与库特征的相似度匹配：

1. 支持 `face` 与 `person` 两种模式（通过 `mode` 标识）。
2. 使用 L2 归一化 + 余弦相似度（向量点积）进行排序。
3. 支持 `topk` 截断、阈值判定与批量查询。
4. 接口形式为模块级函数，不再使用类封装。

---

## 2. 代码位置

- `src/tools/feature_matcher.py`

---

## 3. 核心类型

### 3.1 单条结果

`MatchItem`

1. `rank: int`（1 开始）
2. `index: int`（命中的 gallery 行号）
3. `score: float`（余弦相似度）
4. `is_match: bool`（是否通过阈值）
5. `gallery_id: Optional[str]`（可选业务 ID）

字段语义：

1. `rank`：当前结果在 TopK 中的名次（从 1 开始）。
2. `index`：命中的 gallery 特征在 `gallery_features` 中的行号。
3. `score`：相似度分数（余弦相似度，越大越相似）。
4. `is_match`：阈值判定结果；当 `threshold=None` 时恒为 `True`。
5. `gallery_id`：可选业务标识；提供 `gallery_ids` 时取 `gallery_ids[index]`，否则为 `None`。

### 3.2 查询结果

`MatchResult`

1. `mode: FeatureMode`
2. `topk: int`
3. `total_gallery: int`
4. `items: List[MatchItem]`

字段语义：

1. `mode`：匹配模式，取值为 `FeatureMode.FACE` 或 `FeatureMode.PERSON`。
2. `topk`：本次实际返回的结果数（会被限制在 `[1, total_gallery]`）。
3. `total_gallery`：gallery 总条数，即 `gallery_features.shape[0]`。
4. `items`：按 `score` 从高到低排序的 `MatchItem` 列表。

---

## 4. 接口定义

### 4.1 单查询匹配

```python
match(
    query_feature: np.ndarray,
    gallery_features: np.ndarray,
    mode: FeatureMode,
    topk: int = 5,
    threshold: Optional[float] = None,
    gallery_ids: Optional[Sequence[str]] = None,
) -> MatchResult
```

说明：

1. `query_feature` 必须是 1 维向量。
2. `gallery_features` 必须是 2 维矩阵，形状为 `(N, D)`。
3. 若 `threshold` 为 `None`，`is_match` 恒为 `True`。
4. 若传入 `gallery_ids`，长度必须与 `N` 一致。

### 4.2 批量查询匹配

```python
match_batch(
    query_features: np.ndarray,
    gallery_features: np.ndarray,
    mode: FeatureMode,
    topk: int = 5,
    threshold: Optional[float] = None,
    gallery_ids: Optional[Sequence[str]] = None,
) -> List[MatchResult]
```

说明：

1. `query_features` 必须是 2 维矩阵，形状为 `(M, D)`。
2. 返回长度为 `M`，每个元素对应一个 `MatchResult`。

---

## 5. 输入校验与异常

`match` 在以下情况会抛出 `ValueError`：

1. `query_feature` 不是 1 维。
2. `gallery_features` 不是 2 维。
3. 查询维度与库特征维度不一致。
4. `gallery_ids` 长度与库特征行数不一致。

边界行为：

1. 当 `gallery_features` 为空（`N=0`）时，返回空结果，不抛异常。
2. `topk` 会被自动限制在 `[1, N]` 范围内。

---

## 6. 输入输出汇总表

| 接口 | 输入 | 输出 | 说明 |
|---|---|---|---|
| `match` | 单条查询向量 + 库特征矩阵 | `MatchResult` | 返回 TopK 排序结果 |
| `match_batch` | 查询矩阵 + 库特征矩阵 | `List[MatchResult]` | 逐行调用 `match` 并保序返回 |

---

## 7. 最小示例

```python
import numpy as np
from src.tools.feature_extractor import FeatureMode
from src.tools.feature_matcher import match

query = np.array([0.1, 0.3, 0.5], dtype=np.float32)
gallery = np.array(
    [
        [0.1, 0.2, 0.6],
        [0.0, 0.3, 0.4],
        [0.9, 0.1, 0.0],
    ],
    dtype=np.float32,
)
ids = ["img_001", "img_002", "img_003"]

result = match(
    query_feature=query,
    gallery_features=gallery,
    mode=FeatureMode.FACE,
    topk=2,
    threshold=0.3,
    gallery_ids=ids,
)

for item in result.items:
    print(item.rank, item.gallery_id, item.score, item.is_match)
```
